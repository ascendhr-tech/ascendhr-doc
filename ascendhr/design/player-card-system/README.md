# Player Card System - UI Mockups

**Epic:** 0.4 - Player Card System  
**Created:** December 16, 2024

## Quick Start

1. Open `01-employee-list.html` in a browser
2. All files link together - click buttons to navigate
3. Use sidebar to jump between features

## Files

| File | User Stories | Description |
|------|--------------|-------------|
| [01-employee-list.html](01-employee-list.html) | US-0.4.2 | DataGrid with search, filters, pagination |
| [02-create-employee.html](02-create-employee.html) | US-0.4.1 | Form with FM-style attribute sliders |
| [03-employee-detail.html](03-employee-detail.html) | US-0.4.3, 0.4.4, 0.4.5, 0.4.7 | Player card view, status modal, roles |
| [04-player-gallery.html](04-player-gallery.html) | US-0.4.8a, 0.4.8b | Card grid, selection, compare modal |
| [05-bulk-import.html](05-bulk-import.html) | US-0.4.6 | CSV upload wizard with validation |

## Navigation

```
Dashboard → Employee List → Create Employee → Success
                         → Employee Detail → Status Modal
         → Player Gallery → Compare Modal
                         → Employee Detail
         → Bulk Import → Validation → Success
```
